## 📖 About

IntellMeet is a full-stack AI-powered meeting platform that enables users to create and join meetings, schedule future meetings, collaborate in real time, manage meeting tasks, and receive AI-generated meeting transcription.

---

## ✨ Features

- 🔐 JWT Authentication
- 📹 Create & Join Meetings
- 🤖 AI Meeting Transcription
- 📅 Meeting Scheduler
- ✅ Task Management
- 🔔 Real-time Notifications
- 👥 Live Participant Tracking
- ⚡ Socket.IO Integration
